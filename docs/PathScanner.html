PathScanner

Class representing an SVG Path Lexer

Constructor

new PathScanner(path)

Create a lexer for SVG path.

WARNING: You shouldn't use this lexer directly, you should prefer using the PathParser class.

The SVG path syntax can be summarized into 5 unary ScannerPathToken types.

  • command: A single command character.
  • comma_wsp: A comma follow by one or more white space character.
  • number: A valid number (either a coordinate or a flag).
  • unknown: Anything that is undefined within the SVG path grammar or an unexpected character belonging to another token.
  • wsp: One or more white space caracters.
